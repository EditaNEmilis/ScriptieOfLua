# ROBLOX

**Roblox** is a multiplayer game, most games here runs with Lua scripting.

## Luau

Roblox's Lua language for game scripting is called **Luau**.

**Luau** is the scripting language that creators use in *Roblox Studio*. It is a fast, small, safe and gradually type embeddable system scripting language derived from the Lua 5.1 version.

## Examples

*'Hello World!' printing*:

```luau
-- Basic and simple printing function
print("Hello World!")
```

*Loop number*:

```luau
-- Print by 10 times from looping by using i
for i = 1, 10 do
    print(i)
end
```

### Script/Local

![Script](https://robloxapi.github.io/ref/icons/light/Script.png)

**Script** is for only the *serverside* for ingame function. *(eg. Parts, Decals)*

![LocalScript](https://robloxapi.github.io/ref/icons/light/LocalScript.png)

**LocalScript** is for only the *clientside* for ingame players. *(eg. Character, Player camera)*
