# GARRYS MOD

**Garry's Mod** (**GMoD**) is a multiplayer game, modification of Half Life 2 and the Source SDK.

## Lua

Implemented for *Garrys Mod 13*, can be used to script mods (eg. SWEPS, gamemodes, and such..)

### States (Realms)

![Client](https://files.facepunch.com/wiki/files/19952/8d7b58bc25e14dd.png)

**Client** state is for the game client by taking the player input and sending it to the server.

![Server](https://files.facepunch.com/wiki/files/19952/8d7b58d7428c9c6.png)

**Server** state is for the game server, either be on the same system as the game client or as a dedicated server.

![Menu](https://files.facepunch.com/wiki/files/19952/8d7b58bce4f33d1.png)

**Menu** state is soley only for the main menu.

![Shared](https://files.facepunch.com/wiki/files/19952/8d7b58d999caa0e.png) | ![ClientMenu](https://files.facepunch.com/wiki/files/19952/8d7b58a905836e9.png) | ![SharedMenu](https://files.facepunch.com/wiki/files/19952/8d7b58e58180414.png)

**Other** are for shared states, can be executed in either one by function or hook.

* Shared (Client and Server)
* Client and Menu
* Shared and Menu (all states - Client, Server and Menu)
